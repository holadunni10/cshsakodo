# School Images Folder

Place general school photos here.

## Required filenames:
- `school-building.jpg`  → Main school building (used in About section hero)
- `logo.png`             → School logo (used in nav and footer, transparent background preferred)
- `hero-bg.jpg`          → Hero section background image (1920×1080 recommended)

## Recommended specs:
- Logo: 200×200px PNG with transparent background
- School building: 800×600px JPG
- Hero background: 1920×1080px JPG
