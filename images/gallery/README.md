# Photo Gallery Folder

Place school event and activity photos here.

## Naming convention:
Use descriptive names like:
- `assembly-2024.jpg`
- `sports-day-2024.jpg`
- `graduation-2024.jpg`
- `science-lab.jpg`
- `classroom.jpg`
- `cultural-day.jpg`
- `waec-2024.jpg`
- `inter-house-sports.jpg`

## The gallery on the website shows up to 12 images.
Add as many as you like — the first 12 will display.

## Recommended specs:
- Size: 800×600 pixels (landscape preferred)
- Format: JPG
- File size: under 300KB per image
