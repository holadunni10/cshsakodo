- `principal.jpg`       → Mr. Awoniran J.A. (Principal)
- `vice-principal.jpg`  → Mrs. Oyegoke (Vice Principal)
- `hod-sciences.jpg`    → Mrs. Ojo (HOD Sciences)
- `hod-humanity.jpg`    → Mr. Onakoya (HOD Humanity)
- `mrs-tajudeen.jpg`    → Mrs. Tajudeen (Business)

## Recommended image specs:
- Size: 400×400 pixels (square)
- Format: JPG or PNG
- File size: under 200KB for fast loading

If a photo is missing, the website will automatically show a placeholder avatar.
