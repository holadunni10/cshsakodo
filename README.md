## 🚀 How to Host on GitHub Pages

1. Create a GitHub account at https://github.com (if you don't have one)
2. Click **"New Repository"** — name it something like `cshs-website`
3. Upload ALL files maintaining the folder structure above
4. Go to **Settings → Pages**
5. Under **Source**, select **"Deploy from a branch"** → `main` → `/root`
6. Click **Save** — your site will be live at:
   `https://YOUR-USERNAME.github.io/cshs-website`

## 🖼️ Adding Photos Later

### Staff Photos (Teacher Pictures)
1. Take or scan teacher passport photos
2. Rename each file exactly as listed above (e.g., `principal.jpg`)
3. Upload to the `images/staff/` folder on GitHub
4. The website will automatically display them — no code changes needed!

### Gallery Photos
1. Take photos of school events, classrooms, sports, etc.
2. Name them as listed (e.g., `sports-day.jpg`, `assembly.jpg`)
3. Upload to `images/gallery/`
4. They will appear in the Gallery section automatically

### School Logo
1. Save your logo as `logo.png` (transparent background preferred)
2. Upload to `images/school/`
3. It will appear in the navbar and footer
